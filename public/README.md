# Public Folder Assets

Place your custom cocktail and seasonal menu images inside this folder (`/public`).

Any files placed here are served as static assets at the root of the web application.

For example:
- If you place `velvet_burn.jpg` in this folder, you can reference it in your cocktail configurations as `/velvet_burn.jpg`.
- If you place `autumn_notes.png` here, reference it as `/autumn_notes.png`.

## How to upload your images in Google AI Studio:
1. Locate this **`public`** folder in the file explorer panel on the left side of your screen.
2. If your browser supports it, you can drag and drop your image files directly into this `public` folder inside the file explorer.
3. Alternatively, you can upload them to the project root and then ask me to organize/move them into this folder under whichever names you choose!
